Software

  • Mil-Pac Std-Barc - time-tested software for generating DoD compliance. Std-Barc encodes MIL-STD-129 compliant RFID labels. It prints MIL-STD-129 Unit, Intermediate and Exterior Container Labels, Military Shipping Labels (MSL's) and MIL-STD-130 Unique Identifier (UID) Labels.
  • Mil-Pac Load Manager - works with Std-Barc to generate WAWF Advance Shipment Notices (ASN's ) for RFID labels created by Std-Barc. It also supports ASN creation for UID containers and palletized loads. With a Value Added Network (VAN), RFID Load Manager uploads ASN data to WAWF.
  • Mil-Pac DD Master - automated contract-driven shipment manager for WAWF Receiving Report and Invoice generation. Designed specifically to support contracts requiring multiple shipments. Includes a contract status report generator, WAWF and Receiving Report forms. Also includes support for WAWF and UID Registry transaction generation and upload via Value Added Network (VAN).
